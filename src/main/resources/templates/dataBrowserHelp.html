Downloading data with the ImmPort data browser

  • Once the Aspera Connect Client is installed the ImmPort data browser will briefly display the Aspera banner with the message "Aspera Connect is running".
  • You are required to login using your ImmPort account credentials and accept the NIAID user agreement to begin using the data browser


  • Once logged in you will be able to browse and download all of the public ImmPort data. The first three columns of the data browser perform three distinct functions
  • The first column containing the gray checkmark allows you to download multiple files and/or directories at once. Select the items you would like and select "Download".
  • The second column containing the page and folder with blue arrow icons allows you to download a single file or directory. Select the item you would like and the download will begin immediately.
  • The third column containing the blue directory names allows you to see the contents of a directory.




  • Upon your first download you will be prompted to confirm the connection when you begin


  • When the download begins the Aspera Connect Transfers window will open. You can open the Transfer Monitor window to get more information by selecting the line-plot icon


  • The target directory of your download is not configured in Firefox. The target directory is configured by selecting the cog icon at the bottom of the Aspera Connect Transfers window and selecting the second Transfers tab
