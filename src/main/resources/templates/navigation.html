Navigation

The ImmPort home page presents study access or study search options.



Study Filter, a search feature, presents a comprehensive search display for refining study data searches. As category selections are made counts for each reflect the available number of studies fitting the criteria.



The free-text search field is combined with Auto-Suggest which begins returning results as you type. Possible search-term matches are displayed in a drop-down list. While ImmPort does not strictly support ID searches, string matching with PubMed IDs will return studies with that specific numeric string.
