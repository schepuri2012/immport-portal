Data Templates

  1. Which Data Upload Templates do you need? Please contact us by email at BISC_Helpdesk@niaid.nih.gov
  2. An online description of the upload templates is available to explain the format and content of the templates and the order with which to upload the templates.
  3. The complete set of ImmPort Upload Templates and the User Guide is available as a zip file.
  4. Descriptions and specifications of the ImmPort Upload Templates are available as a pdf file.
  5. Example data upload packages are available for review.
  6. An interactive graphic of the ImmPort Data Model is available
  7. A set of decision tree diagrams is available here.
  8. For transcription profiling data sets, the HIPC Standards Working Group Transcription Profiling standards are recommended.
  9. For flow cytomtery, mass cytometry or CyToF data sets, please see the standards recommendation of the HIPC Bioinformatics Standards Working Group Cytometry guidelines.
  10. Please save spreadsheet .xlsx templates as tab delimited .txt files and upload the txt file version.
  11. You may upload one or more files or a folder (directory) of files.
  12. An example of a data provider's open source ImmPort upload preparation utility is available at the GitHub project Immpload
